# zeppelin-parser

Simple dependency parser for zeppelin notebooks

## Usage

    lein run -m zeppelin-parser.core my-notebook.json

or

    lein uberjar
    java -jar target/zeppelin-parser-0.1.0-SNAPSHOT-standalone.jar my-notebook.json
